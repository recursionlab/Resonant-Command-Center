# THE ALGEBRA OF DISAMBIGUATION

**Kory Ogden** — May 2026

---

## ABSTRACT

A formal, executable, and empirically testable framework for modeling how meaning arises from recursive disambiguation dynamics. The core structure is a bipartite graph of concept nodes and mediating nodes, from which we construct a non-associative concept algebra. An explicit Ψ-calculus defines state evolution via collapse, propagation, and stabilization. Categorical reconciliation is achieved via pushouts over shared observational interfaces, with irreducible contradiction captured as an obstruction class.

**Keywords:** non-associative relation algebras, knowledge graphs, categorical pushouts, Ψ-calculus, zero divisors, contradiction as obstruction.

---

## 1. THE RELATIONAL SUBSTRATE

Let:
- **A** = set of article nodes (concepts)
- **D** = set of disambiguation nodes (mediators)
- **E ⊆ (A×D) ∪ (D×A)** = directed edges

The graph G = (A ∪ D, E) is bipartite: edges only connect articles to disambiguation pages and vice versa.

**Mediated reachability:** For x,y ∈ A, define x ⇝ y iff there exists d ∈ D such that (x,d), (d,y) ∈ E. This is the fundamental unit of disambiguation.

---

## 2. THE CONCEPT ALGEBRA

Let C = P(A) be the power set of articles.

**Relative product:** For X,Y ⊆ A:
**X · Y := {y ∈ Y | ∃x ∈ X, x ⇝ y}**

The product is **non-commutative** and generally **non-associative**.

**Closure product:** X ⋃ Y = ∪_{n≥1} X ·^{(n)} Y

**Properties:**
- Distributivity: (X∪Y)·Z = (X·Z)∪(Y·Z)
- Zero element: ∅·X = ∅ = X·∅
- Non-associativity in general: ∃X,Y,Z with (X·Y)·Z ≠ X·(Y·Z)

**Zero divisors:** A pair (X,Y) with X≠∅, Y≠∅ is a zero divisor if X·Y = ∅.

---

## 3. MEASURABLE INVARIANTS

**Associativity defect:** α = Pr(({x}·{y})·{z} ≠ {x}·({y}·{z}))

**Zero divisor density:** ρ = Pr({x}·{y} = ∅ | x,y ∈ A)

**Closure depth:** β(X) = min{k | X^{(k)} = X^{(k+1)}}

These invariants are computed via Monte Carlo sampling for large graphs.

---

## 4. THE Ψ-CALCULUS

**State:** A subset X ⊆ A.

**Operators:**
- **Propagation:** prop(X) = X ∪ {y ∈ A | ∃x ∈ X, x ⇝ y}
- **Collapse (attractor):** Attr(X) = SCC(X ∪ ε(X)) where SCC is strongly connected component closure
- **Evolution:** Ξ(X) = prop(Attr(X))
- **Stabilization:** Ψ(X) = lim_{k→∞} Ξ^k(X) = Fix(Ξ)

**Dynamic glitch:** Glitch_dyn(X) = X \ ∪_{k≥1} Ξ^k(X) — nodes that disappear under iteration and never return.

---

## 5. CATEGORICAL RECONCILIATION

**Theory category:** Objects are theories, morphisms are structure-preserving translations.

**Observation functor:** Obs: 𝒯 → 𝒞, mapping a theory to its observable support.

**Reconciliation as pushout:** Given a span T₁ ← I → T₂ where I is a shared observational interface, the reconciled theory is T* = T₁ ⊔_I T₂.

**Reconciliation state:** X* = Ψ(X₁ ∪ X₂)

**Obstruction class:** Ω := X* \ (X₁ ⋆ X₂)

Then: X* = (X₁ ⋆ X₂) ∪ Ω

- **X₁ ⋆ X₂** = consistent core (all identifications forced by direct compositional closure)
- **Ω** = irreducible mismatch (nodes that appear only after recursive propagation)

> "Contradiction is not a failure; it is a stable invariant of the reconciliation process."

---

## 6. TOPOLOGICAL DIAGNOSTICS

**Alexandrov topology:** On vertex set V = A ∪ D, declare U ⊆ V open if it is upward closed under directed reachability.

**Nerve:** The free category generated by the directed graph has a nerve that is a simplicial set. The geometric realization |N(G)| is a finite cell complex.

**Hurewicz map:** h: π₁(|N(G)|) → H₁(|N(G)|). A non-trivial kernel indicates non-abelian cycles in the reachability structure — persistent logical loops that Ψ-dynamics might not resolve.

---

## 7. TOY EXAMPLE

A = {a,b,c}, D = {d₁, d₂}
Edges: a→d₁→b, b→d₂→c

Mediated reachability: a⇝b, b⇝c

Product: {a}·{b} = {b}, {b}·{c} = {c}
({a}·{b})·{c} = {c}, {a}·({b}·{c}) = ∅

**Associativity fails: α > 0**

Ψ-dynamics from X = {a,c}: Ξ({a,c}) = {a,b,c} → fixed point. Glitch = ∅.

Obstruction for T₁={a}, T₂={c}: X₁ ⋆ X₂ = ∅, X* = {a,b,c}, Ω = {a,b,c}. All nodes are obstructions — the theories are reconciled only after recursive propagation discovers the mediating chain a→b→c.

---

## 8. IMPLEMENTATION

For large graphs (|A| ≈ 10⁴):
- Computing Ξ(X) requires O(|X| · deg_max) time
- Stabilization reaches fixed point in O(diameter) steps
- SCC computation is O(|V| + |E|) with Tarjan's algorithm
- Sampling for α, ρ uses small subsets (size ≤ 3) and Monte Carlo

The framework scales to millions of nodes with appropriate sampling and caching.
